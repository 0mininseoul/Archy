# Archy Strategic Review Context

업데이트: 2026-03-06
목적: 데일리 전략 리뷰 생성 시 필요한 최소 맥락만 안정적으로 제공한다.

## 제품 한 줄 정의
Archy는 모바일/PWA에서 음성을 녹음하면 자동으로 문서를 만들고 Notion/Google/Slack으로 연결해주는 서비스다.

## 핵심 사용자 가치
- 기록 자동화: 회의/강의/인터뷰 후 정리 시간을 줄인다.
- 모바일 연속성: 이동 중/네트워크 불안정 상황에서도 세션 복구와 재시도로 완료율을 높인다.
- 공유 자동화: 생성된 결과를 외부 툴로 바로 전달해 후속 작업 전환을 빠르게 만든다.

## 현재 운영 관점 핵심 지표
- 총 유저 수
- 가입전환율
- 온보딩율
- PWA 설치율
- 연동율
- 활성화율(30일)
- 결제율
- 헤비 유저 상위 목록

해석 원칙:
- 절대 수치보다 전일 대비 변화와 병목 원인을 우선 해석한다.
- 지표 간 연결(예: 가입전환율 상승 vs 온보딩 정체)을 함께 본다.

## 제품/비즈니스 제약
- 모바일 우선 UX를 유지한다.
- 데이터 보안/권한 분리를 유지한다.
- 실패 복구/재시도 가능한 운영 흐름을 우선한다.
- 실행 액션은 "내일 바로 할 수 있는 수준"이어야 한다.

## 전략 리뷰 출력 원칙
- 길게 설명하지 말고 근거 중심으로 압축한다.
- 액션은 필요한 만큼만 제시한다(1~5개).
- 각 액션은 기대효과를 한 줄로 붙인다.
- 데이터가 부족하면 추정 대신 확인 요청을 명시한다.

## 제외 원칙
- 이 문서에 없는 상세 구현 디테일을 추측해 단정하지 않는다.
- 근거가 약한 주장/과장 표현은 피한다.
